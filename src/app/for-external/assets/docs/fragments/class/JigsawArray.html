 set

将位置index处的数据更新为value。JigsawArray不支持采用方括号表达式设置一个值，因此必须通过这个方法来替代。

const a = new ArrayCollection<any>();
a[0] = 123;    // compile error!
a.set(0, 123); // everything is fine.

返回类型 void

  • index: number
  • value: T
