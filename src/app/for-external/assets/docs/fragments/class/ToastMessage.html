普通属性 / Properties

名称类型说明默认值示例
 heightstring | number

提示框的高度，单位是px，默认由内容自动撑开

 iconstring

提示框的图标，目前支持Jigsaw自研的@rdkmaster/icon-font符号图标。默认无图标。

 iconTypeNoticeLevel

提示框支持的默认类型

 timeoutnumber

提示信息超时自动关闭的毫秒数，为0则表示不关闭，默认8000ms

 widthstring | number

提示框的默认宽度，单位是px，默认350px
