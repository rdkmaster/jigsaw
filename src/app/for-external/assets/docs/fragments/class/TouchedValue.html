TouchedValue

起始版本：v1.1.4

用户在界面上，通过渲染器“摸”了表格的附加列之后，表格会将用户当时“摸过”列的相关信息， 以此类型保存在内存中备用。这些数据是渲染器在判定哪些列被“摸过”时的关键数据， 应用也可以通过AdditionalTableData的各个api更新这些数据。
