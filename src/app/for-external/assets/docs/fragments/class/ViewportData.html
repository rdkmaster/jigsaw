ViewportData

起始版本：v1.0.0

一个抽象的视口数据，记录了这些信息：

  • 视口尺寸：width / height
  • 视口所处区域的尺寸：maxWidth / maxHeight
  • 视口左上角位置：horizontalTo / verticalTo

普通属性 / Properties

名称类型说明默认值示例
 heightnumber

视口当前的高度值

 horizontalTonumber

视口的左上角当前处于全局的水平位置，从0开始计数。

 maxHeightnumber

视口所处区域的最大高度值

 maxWidthnumber

视口所处区域的最大宽度值

 verticalTonumber

视口的左上角当前处于全局的垂直位置，从0开始计数。

 widthnumber

视口当前的宽度值
