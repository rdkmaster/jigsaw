JigsawInput

起始版本：v1.0.0

单行输入框组件，常常用于接收用户的文本输入。

支持前后置图标，且每个图标都可交互，参考demo。

这是一个表单友好组件。

方法 / Methods

名称说明返回值参数说明示例
 focus

调用此方法可以通过编程方式使得文本获得焦点。 当确信用户需要在文本框中输入时，自动让文本框获得焦点可以提升体验。

返回类型 void

--
  • focus
 select

调用此方法可以通过编程方式选中文本框中的所有文本。 当确信用户需要修改文本框里的文本时，自动选中所有文本可以提升体验。

返回类型 void
