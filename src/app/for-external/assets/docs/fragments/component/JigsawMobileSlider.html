名称类型默认值说明示例

 marksSliderMark[]

marks 标签 使用格式为 [Object] 其中 Object 必须包含value 及label 可以有style 属性 例如: marks = [{value: 20, label: '20 ℃'},

 max

输入范围的可选最大值.

 min

可选范围的最小值

 step

每次变化的最小值, 最小支持小数点后两位.

 validbooleantrue
 valuenumber | ArrayCollection<number>

slider的当前值, 类型 number | ArrayCollection 支持多触点.

 verticalbooleanfalse

垂直滑动条 默认 false
