JigsawMobileTab

起始版本：v1.0.0

使用JigsawTab来将一组视图叠加在同一个区域使用，并以页签的方式来切换这些视图。 JigsawTab提供了多个api用于动态创建、销毁、隐藏tab页， 这些是利用JigsawTab实现复杂、交互密集的视图的有力工具。

如果需要动态增减的视图内容形式比较单一，也可以通过ng-for来实现tab动态化， 参考这个demo。

输出属性 / Outputs

名称数据类型说明示例
 addJigsawMobileTab

发送add事件，携带tabs的实例

  • editable
 removenumber

删除tab时，发出事件，携带删除的tab索引值

  • editable
 selectChangeJigsawMobileTabPane

当所选的tab页发生变化时发出此事件，事件携带的是被选中的tab页实例， 如果需要索引值，请使用selectedIndexChange事件。

当前选中的tab页编号发生变化时，发出此事件。 事件携带的是索引值，如果需要获取更多信息，请参考selectChange事件。

 titleChangeTabTitleInfo

改变tab标题时发送此事件，事件携带一个TabTitleInfo类型的数据。

 addTab

通过编程的方式添加一个新的tab页

返回类型 any

  • titleString: string

    以一个简单的字符串作为标题

  • contentTemplate: TemplateRef<any>

    以一个ng-template标签包围起来的模板作为tab页的内容， 当tab页的内容比较简单时，建议采用此方式。

  • initData: Object

    提供给contentTemplate的初始化数据

  • activateImmediately: boolean

    是否立即激活新增的Tab页，默认值是true

 hideTab

隐藏对应的Tab页，使用户在界面上看不到它，页无法对它做操作。 注意这个方法不会销毁tab页内的组件，如果需要销毁tab页，请使用removeTab。 被隐藏的tab页可以通过showTab方法让他们再次显示出来。

返回类型 void

  • index: number

    tab页的编号，从0开始

  • api
  • hide-tab
 removeTab

销毁指定的Tab页，注意此操作不可恢复，可以使用hideTab来隐藏一个tab页，而非销毁它。

返回类型 void

  • index

    目标tab页的编号，从0开始计数。

  • destroy-tab
  • api
 showTab

将对应的Tab页切换为激活状态，当指定的tab页是隐藏状态时，它将被设置为正常状态并被激活。

返回类型 void

  • index: number

    tab页的编号，从0开始
