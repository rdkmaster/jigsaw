IWingsTheme

起始版本：v1.0.0

Jigsaw提供了一套非常功能强大的皮肤系统

  • 支持多种Ux规范的皮肤
  • 支持深浅色系皮肤的在线热切换
  • 支持局部区域内使用相反色系皮肤
  • 同时还提供了一套css变量用于帮助应用创建支持上述各个功能的页面 详细的使用方法请参考 这个PR 和 这个PR。
