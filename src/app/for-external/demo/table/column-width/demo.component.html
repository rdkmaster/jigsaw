自定义列宽用法
  • contentWidth默认为'auto'，默认按照表格宽度平均分配列宽；可以通过columnDefine设置单个列宽，当单个列宽设置成'byContent'时，会根据列的内容计算宽度
  • 当contentWidth为'_inner_auto_'时，会根据表头内容自动伸展，可能会自动产生横向滚动条，但无法设置列宽，列宽没有响应性，不推荐使用；
  • contentWidth也可以直接设置横向的总宽度，分配宽度规则与auto一样，即按照表格宽度平均分配列宽；
