TimeService

起始版本：v1.0.0

提供了日期运算、格式化、转换等一系列常用的方法，当你需要对时间做任何操作时，它都可以帮到你。

 convertValue

将一个时刻转成符合改粒度的值，例如

TimeService.convertValue("now", TimeGr.date); // -> 2018/3/23 00:00:00

"now"的时分秒被切去。

返回类型 string

  • value: WeekTime

    待转换的时刻，支持时间宏，请参考这里TimeUnit的说明。
